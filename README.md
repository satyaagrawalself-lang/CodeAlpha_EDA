# Exploratory Data Analysis on Book Dataset

## Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on a book dataset collected through web scraping. The objective is to understand the structure of the dataset, analyze book prices and ratings, and extract meaningful insights.

## Tools and Technologies Used

* Python
* Pandas
* Google Colab

## Dataset Information

The dataset contains:

* Book Title
* Price
* Rating

## Analysis Performed

* Data inspection and structure analysis
* Missing value detection
* Statistical summary of book prices
* Rating distribution analysis
* Identification of the most expensive and least expensive books

## Key Findings

* The dataset contains 20 book records.
* No missing values were found.
* The average book price is 38.05.
* The highest book price is 57.25.
* The lowest book price is 13.99.
* One-star rated books are the most common category.

## Conclusion

The dataset is clean and suitable for further analysis and visualization. The project demonstrates the use of Python and Pandas for data exploration and insight generation.
